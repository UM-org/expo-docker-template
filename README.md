# expo-docker-example

An example of Expo React Native with docker
## Prerequisites : 
    - Docker Desktop
    - docker-compose
    - expo-cli

## Steps :
    - `npx expo-cli init react_native_app --npm`
    - `docker-compose build`
    - `docker-compose up`
